What is Steganography?

Steganography is the method of hiding information inside another file, such as an image, audio file, or video.

How This Demo Works

  • Your message is converted into binary numbers.
  • The binary data is placed into image pixel values.
  • The changed image is downloaded as a PNG file.
  • The message can later be extracted from that PNG image.

Why Use Encryption?

Encryption changes readable text into unreadable text. Only someone with the correct password or key should be able to read it again.
